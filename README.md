# READ ME

Create new python enviornment:

```shell
python3 -m venv myvenv
```

Activate python virtual enviornment:
```shell
cd myvenv
source /workspaces/myRepository/myvenv/bin/activate
```

Deactivate virtual enviornment to return to home directory:
```shell
deactivate
cd ~
```
